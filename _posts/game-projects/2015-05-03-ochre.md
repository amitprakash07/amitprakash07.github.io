Ochre is the game selected by the industry panels to proceed as a thesis project. The game has a unique controlling mechanics which makes it different. The main mechanics of the game is to use a web camera as the only controller for the player. The player needs to control a physical card which upon recognition does a unique behavior as implemented in the game.


![Ochre screenshot]({{ '/assets/img/blog/2015/maui-ochre/ochre-title.jpg' | relative_url }})

Initially, we struggled a lot in selecting technology for achieving the goal. Firstly, we tried with [**OpenCV**](http://opencv.org/). It was working good with object recognition but it was not meant for tracking the object using the camera.

Our producer, got a hint of SDK calle Metaio which gives tracking facilities. After playing around the [**Metaio SDK**](http://www.metaio.com/) we were able to recognize and track templated objects from the physical card. 

> **Note:** Metaio was later acquired and its SDK is no longer maintained; this post reflects the **2015** toolchain. The same line of work later continued as **Maui**.
