# Blog posts

Place dated Markdown files here in subfolders (Jekyll still picks them up).

- **`game-engineering/`** — Game engine / runtime coursework (the `*game-engineering-*` post series). Titles use **`Game engine coursework:`**.
- **`game-projects/`** — Studio-style projects, jams, thesis-era work, and related posts. Titles use **`Project coursework:`**. (Later you can add e.g. **`computer-graphics/`** with **`Computer graphics coursework:`** for a CG strand.)

Filename must stay **`YYYY-MM-DD-slug.md`**. Permalinks follow `_config.yml`: **`/blogs/game-engineering/YYYY/MM/DD/slug/`** — the first URL segment matches this folder (`game-engineering`, `game-projects`, …).
