# amitprakash07.github.io

## Run locally (Windows / PowerShell)

From this repo folder:

```powershell
bundle -v
```

If Bundler isn't installed:

```powershell
gem install bundler
```

Install gems and run the site:

```powershell
bundle install
bundle exec jekyll serve --livereload
```

Open `http://127.0.0.1:4000/`.
